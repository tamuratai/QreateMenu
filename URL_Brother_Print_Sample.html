Important:



Printing from other models:

  1. Connect your mobile device to the printer.
  2. Share the template file (LBX file) and media information file with Smooth Print, and then type the filenames in the corresponding sample program fields.
  3. If you need to replace any objects, make sure that the object names are the same as those used in the sample program.

For more information, see the webpage.

事前準備：



他のモデルでの印刷：

  1. 接続：モバイル端末とプリンターを接続します。SmoothPrintで接続するプリンターを設定します。
  2. 印刷：テンプレートファイル (.lbx) と用紙情報ファイル (.bin) をSmooth Printに共有し、テンプレートファイルのファイル名を "filename" 入力ボックスに、用紙情報ファイルのファイル名を "size" 入力ボックスに入力します。
  3. オブジェクトの置き換え：オブジェクト置き換え機能を使用する場合、テンプレートファイルで設定したオブジェクト名がサンプルプログラムと同じであることを確認します。

Smooth Printの詳細な使用方法はこちらをご参照ください。
